HTML

  • The head element contains information about the webpage. This includes things like the language, website title, character set, and more.
  • The body element represents the visible content shown to the user. This includes text, images, graphic elements, etc.
  • Sections of content within the HTML body can be identified by CSS, JavaScript, etc., by assigning "classes" (class=) and "IDs" (id=)
  • HTML essentially acts as the building blocks or skeleton of a website. CSS assigns styles to HTML content, and JavaScript gives it function.
  • ...

CSS

  • CSS (Cascading Style Sheets) effectively assigns styles to content you've coded in HTML.
  • CSS defines colors, aesthetics, visual layout, and organization.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS can be written directly inline in your HTML for permanent changes made to specific sections/parts of HTML.
  • CSS can be written as an internal stylesheet, making style changes only to HTML on the current page/HTML file.
  • For most applications, CSS can be written in an external style sheet and linked into the head of your HTML, making changes to anything within the HTML and written in a separate .css file.
  • When writing CSS, you can add properties to HTML items directly by name (header, footer, h1, ul, p, etc.) or by calling a class (i.e: .card{})
  • Various modifiers can be applied to your HTML code, including height/width, margins, padding, border, shadow, color, justification, and much more.

Git

  • cd directory-name: changes to target directory
  • cd ..: backs out one directory level
  • ls: lists contents of current directory
  • touch file-name: creates new file of specific type
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin branch-name: syncs latest version of branch to local system
  • git status: identifies current branch and whether or not it is up to date
  • git add -A: commits all files in the working branch
  • git commit -m "description of changes": commits changes in branch to repository, with the "-m" flag allowing us to associate a message with the commit.
  • git push origin branch-name: pushes committed changes to repository
  • ...

JavaScript

  • JavaScript gives function/interactivity to your HTML code.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script. JavaScript executes code from top to bottom.
  • JavaScript adds functionality to web apps by calling functions containing code blocks that execute different tasks using assigned variables.
